Projects

In collaboration with two team members, I spearheaded the development of an e-voting platform that mirrors Pakistan's election system using Python. Central to this project was the implementation of a user interface utilizing the Tkinter library to ensure a seamless voting experience. We meticulously enforced eligibility criteria, restricting access to citizens aged 18 and above, aligning with Pakistan's age requirement for voting. Moreover, we tailored the voting process to the user's city, mimicking the constituency-based voting prevalent in Pakistan. To manage secure user data storage, we utilized the PyODBC library, drawing parallels to Pakistan's NADRA database infrastructure. Additionally, we integrated data visualization features using Matplotlib, enabling the presentation of election results through informative graphs. Notably, we designed the system with scalability in mind, laying the groundwork for potential future integration of constituency-based voting, contingent on data availability. Through our collaborative efforts, we aimed to create a robust and reliable platform that mirrors the intricacies of Pakistan's electoral system.
